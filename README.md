## 🛠️ 기술 스택

- **Backend:** Nest.js, TYpeScript  
- **Database:** PostgreSQL, Prisma, AWS RDS/S3
- **API 문서화:** Swagger
- **테스트:** Jest,SuperTest, ESLint
- **협업 도구:** Git & GitHub, Discord, Notion
- **배포/운영:** AWS EC2, Nginx, PM2
- **CI/CD:** Gitgub Actions
